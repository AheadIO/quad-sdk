Detailed Description

A class for interfacing between dummy robot data and other spirit-software nodes and topics.

RobotProxy contains the approximation for the real spirit robot. Its purposes are purely for development. The class provides publishers for dummy robot data (encoder, imu, and mocap measurements) as well as a subscriber for control inputs.
