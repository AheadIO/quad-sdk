Detailed Description

A class for interfacing between stand-in robot data and other spirit-software nodes and topics.

RobotInterface contains the approximation for the real spirit robot. Its purposes are purely for development. The class provides publishers for robot data (encoder and imu measurements) as well as a subscriber for control inputs.
